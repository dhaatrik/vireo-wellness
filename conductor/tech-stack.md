# Tech Stack: Vireo Wellness

## Frontend
- **React 19:** Core library for building the reactive user interface.
- **TypeScript:** Ensures type safety and provides an excellent developer experience.
- **Vite:** Next-generation frontend tooling providing lightning-fast HMR and optimized builds.
- **Tailwind CSS v4:** Utility-first CSS framework for rapid and consistent styling, configured for the application's dark theme.
- **React Router v7:** Seamless client-side routing.
- **Recharts:** Composable charting library used for dynamic visualizations (e.g., blood sugar trends).
- **Framer Motion:** Powers fluid animations and page transitions.

## Testing & Quality
- **Vitest:** Fast unit test framework.
- **ESLint:** Code style consistency and catching potential issues early.
